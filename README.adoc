= IT Self-Service Agent Quickstart Showroom

Lab content for the *Automate IT Processes with IT Self-Service Agents* quickstart — AI-driven IT process automation using Red Hat OpenShift AI, Llama Stack, LangGraph, and MCP servers.

== Content Repo

This Antora-based showroom provides hands-on lab instructions for:

* Deploying the IT self-service agent platform on OpenShift
* Interacting with the laptop refresh agent via CLI
* Running evaluations and tracing agent workflows

The install and quickstart code lives in https://github.com/rh-ai-quickstart/it-self-service-agent[rh-ai-quickstart/it-self-service-agent]. This showroom uses **Demo Mode** (`make install INSTALL_MODE=demo`).

== Quick Start

. Clone this repo and preview locally:
+
[source,sh]
----
podman run --rm --name antora -v $PWD:/antora:z -p 8080:8080 -i -t ghcr.io/juliaaano/antora-viewer
----
+
(On non-SELinux systems, omit `:z`.)

. Edit content in `content/modules/ROOT/pages/`

== Local Development

*Option 1 — single run:*
[source,sh]
----
podman run --rm --name antora -v $PWD:/antora:z -p 8080:8080 -i -t ghcr.io/juliaaano/antora-viewer
----

*Option 2 — podman-compose (live reload):*
[source,sh]
----
podman-compose up
----

Open http://localhost:8080 after the build completes.

See the https://rhpds.github.io/showroom_template_nookbag/modules/content-repo.html[Content Repository] guide for directory layout and dev mode.
